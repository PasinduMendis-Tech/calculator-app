# Calculator App

A responsive calculator application built with HTML, CSS, and JavaScript.

## Live Demo
https://pasindumendis-tech.github.io/calculator-app/

## Features
- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Decimal number support
- Clear (C) functionality
- Mobile-friendly user interface

## Technologies Used
- HTML5
- CSS3
- JavaScript
- GitHub Pages

## Purpose
This project demonstrates frontend logic, event handling, and responsive UI design using vanilla JavaScript.
